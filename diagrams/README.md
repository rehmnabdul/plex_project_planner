# Diagrams Directory

This directory contains visual diagrams for the Todoist clone project.

## Diagram Types

### ERD Diagrams
- Entity Relationship Diagrams showing database structure
- Entity relationships and cardinalities
- Available in Mermaid format (in planning documents) and visual formats

### Use Case Diagrams
- Use case diagrams showing user interactions
- Actor relationships
- Available in Mermaid format (in planning documents)

### Architecture Diagrams
- System architecture diagrams
- Component diagrams
- Deployment diagrams

## Diagram Formats

### Mermaid Format
- Text-based diagram format
- Can be rendered in Markdown viewers
- Version controlled in planning documents

### Visual Formats
- PNG/SVG formats for presentations
- Can be generated from Mermaid source
- Tools: Mermaid Live Editor, Draw.io, PlantUML

## How to View Diagrams

### Mermaid Diagrams
1. View in planning documents (rendered automatically in many Markdown viewers)
2. Use [Mermaid Live Editor](https://mermaid.live/)
3. Use VS Code with Mermaid extension

### Visual Diagrams
1. Open PNG/SVG files directly
2. Use diagram editing tools to modify

## Generating Visual Diagrams

### From Mermaid
1. Copy Mermaid code from planning documents
2. Paste into [Mermaid Live Editor](https://mermaid.live/)
3. Export as PNG or SVG
4. Save to this directory

### Recommended Tools
- **Mermaid Live Editor**: Online Mermaid renderer
- **Draw.io**: General diagramming tool
- **PlantUML**: Text-based UML diagrams
- **Lucidchart**: Professional diagramming

## Diagram Maintenance

- Keep diagrams synchronized with code
- Update ERD when database schema changes
- Update use case diagrams when features change
- Version control all diagram source files
